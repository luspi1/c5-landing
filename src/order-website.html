Заказать сайт

На этой странице Вы можете заказать сайт для существующего объекта размещения.

Внимание: каждый объект размещения (гостиница, санаторий, бутик-отель и так далее) требует отдельного интернет-сайта, при этом система управления для всех ваших объектов — единая.

Наименование сайта для отображения в адресной строке и заголовке. Будет также показано во всех внутренних списках.

Будет использован в адресе сайта до перехода сайта из тестового в коммерческий режим

Логотип должен обязательно быть в формате SVG, привычные форматы JPG и PNG не подойдут.


Фона быть не должно. Цвет логотипа подбирайте исходя из выбранной цветовой схемы.
